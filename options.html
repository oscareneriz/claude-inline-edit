Claude Inline Edit

Highlight text anywhere → click the ✦ pill (or press Ctrl/Cmd+Shift+K) → tell Claude how to change it.

Style presets

★ marks the preset shown first when the panel opens. Click a star to set it (click again to clear).

Highlight pill

When off, the pill won't appear — just highlight and press Ctrl/Cmd+Shift+K to open the window.
Click “Set pill position”, then switch to a web page: a pill appears in the center with a Save button. Drag it where you want and click Save position. The pill will appear at that spot from then on.

Window size

Resize the popup by dragging its bottom-right corner — the size is saved for whichever mode is active. Use the ⤢ button in the popup header to switch Small / Big. (Opens Small by default.)

Connection

Get one at console.anthropic.com → API keys. Stored only in your browser; sent only to Anthropic.
